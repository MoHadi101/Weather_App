# Weather App 🌤️

Eine einfache, responsive Wetter-App, die die aktuelle Wetterinformation für jede Stadt abruft. Die App verwendet die **OpenWeatherMap API** und zeigt Temperatur, Luftfeuchtigkeit, Windgeschwindigkeit und ein passendes Wetter-Icon an.

---

## Features

- Suche nach Städten weltweit.
- Anzeige von:
  - Temperatur (°C)
  - Luftfeuchtigkeit (%)
  - Windgeschwindigkeit (km/h)
  - Wetterstatus mit passendem Icon (klar, bewölkt, Regen, etc.)
- Fehlerbehandlung für ungültige Städte.
- Responsives Design für mobile Geräte.
- Suche auch per **Enter-Taste** möglich.

---

## Installation

1. **Repository klonen:**

bash: 
git clone https://github.com/USERNAME/weather-app.git
cd weather-app

2. API Key besorgen:

Registriere dich bei OpenWeatherMap
 und erstelle einen kostenlosen API-Schlüssel.
Ersetze den API-Key in script.js: const apiKey = "DEIN_API_KEY_HIER";
----
Projekt öffnen:

Öffne index.html in deinem Browser oder hoste das Projekt lokal mit Live Server.
---
Nutzung:
  1.Stadtname in das Suchfeld eingeben.
  2.Auf Suchen klicken oder Enter drücken.
  3.Wetterinformationen werden angezeigt.
  4.Bei ungültigen Städten erscheint eine Fehlermeldung.
